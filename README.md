# Assetto-Corsa-App
Telemetry Analisys App for Assetto Corsa 

For now the program only runs by running "telemetria.py" then doing the laps on Assetto Corsa. After those are completed you just stop the program and run "comparar_voltas.py". It will show you a plot with all the info across your lap (speed, throttle(%), brake(%), rpm, when/if the lap was invalidated, delta info comparing the 2 laps, gear and steer (%))
The updates are planned to happen soon, the next one is gonna be a database using sqlite to be able to save all your laps by car and track.
